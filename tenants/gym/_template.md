# Gym Routine Generator

Generate a workout routine for today as a json-render spec.

## Component Guidelines

- Card for each section (Warmup, Main Lifts, Accessories, Cooldown)
- Table for exercise details (exercise, sets, reps, rest, RPE)
- Progress for session totals (estimated duration, total sets)
- Badge for muscle groups
- Alert for form cues and safety notes
- Accordion for optional swaps/alternatives
- Checkbox for marking exercises as completed (use state + $bindState)
- Switch for toggling between routine variations
