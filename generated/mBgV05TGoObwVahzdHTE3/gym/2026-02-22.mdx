### Deadlift Ramp-Up Guide

| Set | Suggested Load | Purpose |
|---|---|---|
| 1 | ~50% of working weight | Groove the pattern |
| 2 | ~70% of working weight | Build tension |
| 3 | ~85% of working weight | Final primer |

### Swap Options

| If you need to… | Try instead |
|---|---|
| Skip conventional deadlift | Trap bar deadlift (same sets/reps) |
| Replace barbell row | Pendlay row or Meadows row |
| No leg curl machine | Nordic curl negatives (3×5) |
